Organize z-index values and provide the `ZIndex` enum which is a preset of z-index values to help control the stacking order of components in consumer applications.

Change the z-index value of the below components.

- `Overlay`, `Select`: `1000` (container and content)
- `Modal`: `1100`
- `Toast`: `1200`
- `Tooltip`: `1300`
