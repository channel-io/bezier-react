# Bezier Figma Plugin

Bezier Figma Plugin is a Figma plugin that helps build Bezier design system and increase productivity.

## Usage

### 1. Build

```bash
yarn build
```

### 2. Import

Open the Figma desktop app

```bash
Plugins > Development > Import plugin from manifest
```
